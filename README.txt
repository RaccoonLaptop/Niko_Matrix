                        Niko_Matrix
         Матрица с надписью + утилиты для сети
═══════════════════════════════════════════════════════════════════════

Привет! Меня зовут Ильшат, в сети — Niko.
Это моя программа: экран-заставка в стиле "Матрица" с падающими символами
и вашей надписью в центре. Плюс быстрый доступ к полезным утилитам.

───────────────────────────────────────────────────────────────────────
  УПРАВЛЕНИЕ
───────────────────────────────────────────────────────────────────────

  ESC              — выход
  Alt+Enter        — полный экран
  Alt+T            — вкл/выкл текст
  Alt+N            — редактор надписи (маска, время)
  Alt+O            — обновить окно
  Alt+A            — о программе
  L                — язык (RU/EN)

  Левый край окна  — панель утилит (VPN, DNS, сброс сети, обход DPI)
  Правый край      — панель настроек

───────────────────────────────────────────────────────────────────────
  НАДПИСЬ
───────────────────────────────────────────────────────────────────────

Откройте matrix.py и отредактируйте TEXT_MASK (в начале файла):

  #  или любой символ  =  рисуем в этом месте
  пробел              =  пусто

Можно рисовать буквы, цифры, логотипы — всё, что помещается в сетку.

───────────────────────────────────────────────────────────────────────
  СБОРКА .EXE
───────────────────────────────────────────────────────────────────────

  1. Запустите build_exe.bat
  2. Дождитесь завершения (1–2 минуты)
  3. Готовый файл: dist\Niko_Matrix.exe

  Требуется запуск от имени администратора.

───────────────────────────────────────────────────────────────────────
  СТРУКТУРА ПРОЕКТА
───────────────────────────────────────────────────────────────────────

  matrix.py          — основной код
  images/            — все картинки (иконки, логотипы)
  create_icon.py     — создание icon.ico из Niko_Matrix.png
  convert_cdpi_icon.py — конвертация CDPI иконки
  Niko_Matrix.spec   — конфиг PyInstaller
  build_exe.bat      — скрипт сборки
